## Detection Signatures

### File System Indicators
- ASPX files in web directories with suspicious names

### Process Indicators
- PowerShell execution with suspicious parameters
- Process injection activities
- Credential dumping tools execution
- Screen capture activities

## Mitigation Recommendations
1. Block known C2 infrastructure at network perimeter
2. Implement email security to detect spear-phishing
3. Monitor for web shell uploads and execution
4. Deploy endpoint detection for credential dumping
5. Implement network segmentation to limit lateral movement
6. Regular patching of public-facing applications
7. Security awareness training for spear-phishing recognition
