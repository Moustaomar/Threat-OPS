## Attack Techniques

### Initial Access
- **Phishing**: Malicious emails with RAT payloads
- **Malicious Documents**: Office documents with embedded RAT
- **Exploit Kits**: Web-based exploit kits
- **Social Engineering**: Targeted attacks on specific individuals

### Execution
- **Python Execution**: Direct execution of Python RAT scripts
- **Process Injection**: Injection into legitimate processes
- **DLL Loading**: Loading malicious DLLs
- **Script Execution**: Execution of malicious scripts

### Persistence
- **Registry Modification**: Modifying Windows registry for persistence
- **Scheduled Tasks**: Creating scheduled tasks for persistence
- **Service Installation**: Installing malicious services
- **Startup Programs**: Adding to startup programs

### Defense Evasion
- **Process Hollowing**: Hiding malicious processes
- **Code Injection**: Injecting code into legitimate processes
- **Anti-Analysis**: Evading security analysis tools
- **Obfuscation**: Obfuscating malicious code

### Command and Control
- **Custom Protocols**: Using custom C2 protocols
- **Non-Standard Ports**: Using non-standard ports for communication
- **Encrypted Communication**: Encrypting C2 communication
- **Domain Fronting**: Using domain fronting techniques

### Collection
- **Data Exfiltration**: Stealing sensitive data
- **Screen Capture**: Capturing screenshots
- **Keylogging**: Recording keystrokes
- **File Theft**: Stealing files from victim systems

### Network Indicators

- **Custom Protocols**: Unusual network protocols and ports
- **Encrypted Traffic**: Encrypted communication patterns
- **Domain Fronting**: Suspicious domain resolution patterns

### Endpoint Indicators
- **Python Execution**: Execution of Python scripts
- **Process Injection**: Suspicious process injection activities
- **Registry Modification**: Changes to Windows registry
- **Scheduled Tasks**: Creation of suspicious scheduled tasks
