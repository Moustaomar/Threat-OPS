## Attack Chain Summary
1. Phishing email with malicious .SVG attachment
2. SVG contains embedded Base64-encoded fake web page
3. Fake web page downloads DOCUMENTO_OFICIAL_JUZGADO.hta
4. HTA drops and executes actualiza.vbs
5. VBS writes and runs PowerShell script (veooZ.ps1)
6. PowerShell downloads encoded text (Ysemg.txt) from dpaste
7. Text decoded to ClassLibrary3.dll (downloader/loader)
8. ClassLibrary3.dll downloads MoNtReaL.dll (injector)
9. MoNtReaL.dll injects AsyncRAT (Client.exe) into MSBuild.exe
10. AsyncRAT establishes C2 connection and persistence

## Persistence Mechanisms
1. Registry Run Key: HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Run\
2. Scheduled Task: schtasks /create /f /sc onlogon /rl highest /tn "<filename>" /tr "<fullpath>"

## Target Injection Process
MSBuild.exe (legitimate Microsoft Build Engine process)

## Anti-Analysis Techniques
- Anti-VM checks (WMI queries, process enumeration)

- Reversed strings
- Junk code insertion
- Process name checks
- Kills analysis tools (Taskmgr.exe, ProcessHacker.exe)
