## Behavioral Indicators

### Encryption Behavior
- Encrypts Master File Table (MFT) on NTFS partitions
- Uses Salsa20 encryption algorithm
- Displays fake CHKDSK messages during encryption
- Creates encryption progress counter file

### UEFI Specific Indicators
- Installs malicious EFI application to EFI System Partition
- Bypasses UEFI Secure Boot using CVE-2024-7344
- Modifies EFI boot configuration
- Persistence through UEFI bootkit
